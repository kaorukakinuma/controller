# Controller

+ Dependencies: `lkeyboard, lpthread`
+ Includes: `controller.h`


## How to build

```
$ git clone https://github.com/kaorukakinuma/controller.git
$ cd controller
$ sudo ./mk.sh
#snip
$ ls /home/lib
libcontroller.a
$ ls /home/include
controller.h
```


## How to run a test

```
$ sudo ./test/run.sh
```
